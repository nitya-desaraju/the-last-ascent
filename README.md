# The Last Ascent

Only the last few humans remain. Escape the underworld city before it's too late. Make it to the surface with the truth of what happened and save humanity. But you are just a spark. What can you do? It can be played [here](https://nitya-desaraju.github.io/the-last-ascent/).

## How to Play

Use arrow keys or 'A' and 'D' keys to move. Press 'SPACE' to jump between wires. Don't let the battery run out.
